# Sistema de Tickets — Unidad 4

Aplicación web estática desarrollada en HTML + CSS + JS con Bootstrap.

## Funcionalidades
✔ Crear tickets  
✔ Prioridad (baja/normal/alta)  
✔ Validación en formulario  
✔ Lista dinámica  
✔ Marcar como leído / eliminar tickets  
✔ Filtros por prioridad  
✔ Contador de urgentes  
✔ Persistencia con localStorage
